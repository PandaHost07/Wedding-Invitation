# Music Folder

Tempatkan file MP3 Anda di sini dengan nama: `wedding-music.mp3`

Langkah-langkah:
1. Download file MP3 lagu pilihan Anda
2. Rename menjadi `wedding-music.mp3`
3. Letakkan di folder `music/`
4. Musik akan automatic di-load dari file lokal

**Saran lagu untuk wedding:**
- Traditional Indonesia (Gamelan)
- Soft piano
- String quartet
- Acoustic guitar

Jika tidak ada file lokal, musik akan fallback ke URL remote yang sudah disediakan.
